# Россия 2050: расчётная демографическая платформа

Самодостаточная статическая платформа для GitHub Pages и локального показа без внешнего интернета. Runtime читает только файлы из `docs/`: локальные данные, локальный Plotly, локальный Three.js и локальные ассеты.

## Страницы

- `docs/index.html` — `СКР`: Россия / федеральный округ / субъект РФ, авторский прогноз СКР до 2050, федеральная рамка цели, краткий и подробный режимы.
- `docs/settlement.html` — `Расселение`: расселение и ИЖС, рост малоэтажного расселения, локальный авторский прогноз численности и локальный прогноз городского/сельского СКР на основе положительной премии сельской/пригородной среды, гауссова процесса и структурного тренда.
- `docs/estate.html` — `Усадьба`: 3D-конструктор многопоколенной усадьбы на локальном Three.js; основной дом строится как цельный расширяемый корпус с этажностью 1–4, объёмной стабильной крышей, безопасной расстановкой деревьев, проектными размерными линиями прямо на земле в 3D-сцене, линией высоты дома и отдельным домом для прародителей только по выбору пользователя.
- `docs/capital.html` — `Маткапитал`: сценарная модель реформы федерального семейного капитала, зарплатный эквивалент 2007 года, подход по стоимости комфортного жилья, бюджетные обязательства и локальные Plotly-графики.
- `docs/mortgage.html` — `Ипотека`: аннуитетный расчёт, льготная и рыночная части кредита, бюджетная субсидия.
- `docs/payments.html` — `Выплаты`: сценарий ежемесячных выплат, охват матерей, дополнительные рождения и цена потенциального рождения.

## Локальный запуск

На Windows можно просто дважды кликнуть `START_LOCAL.cmd` в корне проекта. Он запустит локальный сервер из `docs/` и откроет платформу в браузере.

```bash
python -m http.server 8000 --directory docs
```

Открыть: `http://127.0.0.1:8000/index.html`.

## Данные

Авторские файлы из GitHub-репозитория прогноза загружены в `docs/data/author_forecast_source/` и преобразованы в локальные runtime-слои:

- `docs/data/author_tfr_forecast_2050.json`
- `docs/data/author_population_forecast_2050.json`
- `docs/data/settlement_tfr_forecast_2050.json`
- `docs/data/forecast_join_audit.json`
- `docs/data/forecast_join_audit.csv`
- `docs/data/territory_levels.json`
- `docs/data/matcapital_inputs.json`

В runtime нет запросов к GitHub Raw, CDN, удалённым API, внешним шрифтам или внешним изображениям.

## Проверки

```bash
python scripts/check_json.py
python scripts/check_no_external_runtime.py
python scripts/check_russian_ui.py
python scripts/check_data_locality.py
python scripts/check_nav_numbering.py
python scripts/check_settlement_forecast.py
"C:\Program Files\Git\bin\bash.exe" scripts/check_js_syntax.sh
npm install
npm run test:smoke
```

Playwright открывает все шесть страниц, блокирует внешние запросы, делает desktop/mobile-скриншоты, проходит ключевые интеракции и visual QA для картограммы, прогноза расселения, Plotly-графиков Маткапитала, слайдера запуска мер и 3D-усадьбы, включая отсутствие preset-кнопок и микробизнес-модуля, этажность, разрывы в заборе, чекбокс отдельного дома для прародителей, безопасную расстановку деревьев, 3D-размерные линии без сдвига камеры, линию высоты дома, stepper-контролы без дублей чисел, выбор участка в сотках, нормальные габариты окон/двери, ориентацию задних окон, pixel-QA крупного вида дома и стабильность непрозрачного рендера крыши.
