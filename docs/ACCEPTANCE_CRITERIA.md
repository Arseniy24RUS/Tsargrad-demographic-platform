# Критерии приёмки релиза

Проект принимается только при одновременном выполнении всех пунктов.

## Архитектура

- [ ] Публикуемая директория — `docs/`.
- [ ] Есть 6 HTML-страниц: `index.html`, `settlement.html`, `estate.html`, `capital.html`, `mortgage.html`, `payments.html`.
- [ ] Навигация единая на всех страницах.
- [ ] Навигация содержит шесть разделов без отвлекающих числовых префиксов.
- [ ] Runtime-зависимостей от внешнего интернета нет.
- [ ] Все библиотеки лежат в `docs/assets/vendor/`.
- [ ] Все данные лежат в `docs/data/`.
- [ ] Все изображения/логотипы/текстуры лежат в `docs/assets/`.

## Данные

- [ ] Авторский прогноз СКР до 2050 хранится локально.
- [ ] Авторский прогноз численности населения до 2050 хранится локально.
- [ ] Нет runtime-запросов к внешнему репозиторию пользователя.
- [ ] Горизонт данных в runtime не выходит за 2050 год.
- [ ] РПН-2022 используется только как федеральный слой.
- [ ] Региональный слой РПН использует только 2012/2017.

## Интерфейс

- [ ] Все видимые тексты на русском языке.
- [ ] Нет `baseline`, `debug`, `TODO`, `prototype`, `RPN`, `CSV`, `XLSX`, `download`, `fetch`, `GitHub Raw`, `NaN`, `undefined`, `null` в видимом UI.
- [ ] Plotly modebar отключён или полностью русифицирован.
- [ ] Технические методики спрятаны в раскрываемые блоки.
- [ ] Первая страница имеет `Краткий вид` и `Подробный вид`.

## Страница СКР

- [ ] Есть режимы `Россия`, `Федеральный округ`, `Субъект РФ`.
- [ ] В режиме России KPI относятся только к России.
- [ ] В режиме ФО KPI разделены на Россию и выбранный округ.
- [ ] В режиме субъекта KPI разделены на Россию и выбранный субъект.
- [ ] Федеральная цель не навязывается субъекту как индивидуальная нормативная траектория.
- [ ] Разведены показатели `Отставание от цели` и `Требуемый прирост после запуска мер`.

## Страница Расселение

- [ ] Используется термин `Расселение и ИЖС`.
- [ ] Нет главного термина `дезурбанизация` в основном UI.
- [ ] Базовая численность берётся из локального авторского прогноза.
- [ ] Эффект структуры расселения называется оценкой, а не полным прогнозом.

## Страница Усадьба

- [ ] Three.js локальный.
- [ ] OrbitControls локальный.
- [ ] Есть fallback для отсутствия WebGL.
- [ ] Дом выглядит как архитектурная модель, а не набор кубиков.
- [ ] Визуальная площадь соответствует расчётной.
- [ ] Размерные стрелки и подписи показываются прямо в 3D-сцене и не сдвигают камеру при включении/выключении.
- [ ] Дверь и окна имеют реалистичный масштаб; задние окна развёрнуты наружу относительно задней стены.

## Страница Маткапитал

- [ ] Runtime-файл `matcapital_inputs.json` лежит в `docs/data/` и не содержит внешних URL.
- [ ] Графики строятся локальным Plotly без внешних запросов.
- [ ] Есть зарплатный эквивалент 2007 года и подход по стоимости комфортного жилья.
- [ ] Сценарий пересчитывает KPI, таблицу и цену потенциального рождения.
- [ ] Кнопка выгрузки называется `Скачать таблицу`.

## Страница Ипотека

- [ ] Кредит делится на льготную и рыночную часть.
- [ ] Аннуитет является основным режимом.
- [ ] Подпись `Цена потенциального рождения при заданной конверсии`.
- [ ] График матерей и потенциала — stacked bar со штриховкой потенциала.

## Страница Выплаты

- [ ] Горизонт только до 2050 года.
- [ ] Верхний итоговый вывод написан простым управленческим языком.
- [ ] Технические параметры спрятаны в дополнительных настройках.

## Автотесты

- [ ] `python scripts/check_json.py` проходит.
- [ ] `python scripts/check_no_external_runtime.py` проходит.
- [ ] `python scripts/check_russian_ui.py` проходит.
- [ ] `bash scripts/check_js_syntax.sh` проходит.
- [ ] Если доступен Playwright, `npm run test:smoke` проходит.
