# Модуль «Инфраструктура поселений»

Модуль оценивает инженерную и социальную готовность населённых пунктов к малоэтажному семейному расселению. Он связывает координаты, численность, динамику населения и демографическую значимость с расстояниями до инфраструктурных объектов.

## Статус данных

Текущая сборка содержит 155 741 населённый пункт по 85 субъектам РФ. Для каждого поселения сохранены координаты, население 2010 и 2020 годов, число детей, муниципальная принадлежность, тип поселения, агломерационный статус, итоговый индекс, класс готовности и расстояния до девяти инфраструктурных слоёв.

Инфраструктурные признаки извлечены офлайн из локального файла `russia-latest.osm.pbf`. PBF и промежуточные CSV не входят в runtime GitHub Pages; браузер получает только компактные JSON из `docs/data/infrastructure/`.

## Индекс готовности

Индекс считается по шкале 0–100 и включает:

- дороги;
- электроснабжение;
- газификацию;
- водоснабжение;
- водоотведение;
- связь;
- образование;
- медицину;
- повседневные сервисы;
- демографическую значимость.

Инженерная часть собирается из дорог, электроснабжения, газа, воды, водоотведения и связи. Социальная часть собирается из образования, медицины и сервисов. Итоговый индекс объединяет инженерную готовность, социальную доступность и демографическую значимость.

## Управленческая классификация

- A — готово к семейному расселению;
- B — быстрая достройка;
- C — инженерный дефицит;
- D — низкая готовность.

Главный дефицит в паспорте поселения — две самые слабые компоненты индекса. Низкий балл означает зону проверки и приоритизации, а не автоматическое решение об отсутствии объекта: полнота открытых геоданных по газу, воде, канализации и связи отличается по территориям.

## Runtime-ограничения

Страница не обращается к внешним API, CDN, удалённым картам или удалённым шрифтам. Карта строится локальным canvas-кодом, графики — локальным Plotly из `docs/assets/vendor/plotly/plotly.min.js`.
