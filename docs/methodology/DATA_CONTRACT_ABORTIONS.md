# Контракт данных модуля «Аборты»

Модуль показывает динамику и территориальные различия по числу прерываний беременности, а также два расчётных показателя: число прерываний беременности на 1000 женщин 15–49 лет и число прерываний беременности на 100 родов.

## Локальные файлы

- `docs/data/abortions_dashboard.json` — основной компактный JSON для страницы.
- `docs/data/abortions_subjects.geojson` — картографический слой субъектов РФ с присоединёнными последними значениями.
- `docs/data/abortions.sqlite` — нормализованная SQLite-база.
- `docs/data/abortions_summary.csv` — сводная таблица последних значений.
- `docs/data/abortions_join_audit.*` — аудит сопоставления территорий.

## Исходные показатели

| Показатель | Файл | Период | Источник в паспорте выгрузки |
|---|---|---:|---|
| Число прерываний беременности | `data (5).xls` | 2005–2024 | Министерство здравоохранения Российской Федерации |
| Число родившихся без мертворожденных | `data (6).xls` | 1990–2023 | Федеральная служба государственной статистики |
| Численность женщин по возрасту | `data (7).xls` | 1990–2025 | Федеральная служба государственной статистики |

## Расчёты

Численность женщин 15–49 лет рассчитывается суммированием возрастных групп: 15–19, 20–24, 25–29, 30–34, 35–39, 40–44 и 45–49 лет.

```text
Прерывания беременности на 1000 женщин 15–49 лет =
число прерываний беременности / численность женщин 15–49 лет × 1000
```

```text
Прерывания беременности на 100 родов =
число прерываний беременности / число родившихся живыми × 100
```

Индекс абортной нагрузки рассчитывается только как вспомогательный картографический индикатор по субъектам РФ за 2023 год:

```text
Индекс = 0,55 × нормированный показатель на 1000 женщин + 0,45 × нормированный показатель на 100 родов
```

## Ограничения

Федеральный 2018 год восстановлен как опубликованный агрегат `567 183`. Производные федеральные показатели за 2018 год рассчитываются от локальных знаменателей `births_count` и `women_15_49_count`. Региональные значения 2018 года остаются `null`, пока нет сопоставимой региональной выгрузки абсолютных значений.

Показатель на 100 родов рассчитан до 2023 года, потому что загруженный файл числа родившихся заканчивается 2023 годом. Для 2024 года отображаются число прерываний беременности и показатель на 1000 женщин 15–49 лет.
