# Модуль «Аборты»: данные и расчётные показатели

Модуль добавляет в платформу страницу `abortions.html`, посвящённую динамике числа прерываний беременности и двум расчётным индикаторам: числу прерываний беременности на 1000 женщин 15–49 лет и числу прерываний беременности на 100 родившихся живыми.

## Локальные файлы

В runtime платформы используются локальные файлы:

- `data/abortions_dashboard.json` — основная витрина для страницы.
- `data/abortions_subjects.geojson` — картограмма субъектов РФ с присоединёнными последними значениями.
- `data/abortions_summary.csv` — сводная таблица последних значений по территориям.
- `data/abortions.sqlite` — нормализованная база данных модуля.
- `data/abortions_join_audit.json` и `data/abortions_join_audit.csv` — аудит сопоставления территорий.

Исходные показатели рассчитаны из трёх загруженных статистических файлов:

1. `data (5).xls` — «Число прерываний беременности», 2005–2024.
2. `data (6).xls` — «Число родившихся (без мертворожденных) за год», 1990–2023.
3. `data (7).xls` — «Численность постоянного населения — женщин по возрасту на 1 января», 1990–2025.

## Формулы

```text
число прерываний беременности на 1000 женщин 15–49 лет =
число прерываний беременности / численность женщин 15–49 лет × 1000
```

```text
число прерываний беременности на 100 родившихся живыми =
число прерываний беременности / число родившихся живыми × 100
```

В интерфейсе используется формулировка «на 100 родов», поскольку она проще для восприятия; методически знаменатель — число родившихся живыми из загруженной статистической выгрузки.

## Ограничения

- Для числа прерываний беременности последний доступный год — 2024.
- Для показателя на 100 родов последний год — 2023, так как ряд рождаемости в загруженном файле заканчивается 2023 годом.
- Федеральный 2018 год восстановлен как опубликованный агрегат `567 183`; расчётные федеральные показатели на 1000 женщин 15–49 лет и на 100 родов построены от локальных знаменателей 2018 года.
- Региональные значения 2018 года не заполняются: в локальной выгрузке нет сопоставимого регионального слоя абсолютных значений, поэтому региональная картограмма за этот год остаётся ограниченной.
- Новые субъекты РФ оставлены на карте как территории без данных и исключены из рейтинга.
- Отсутствие значения в открытых данных не интерпретируется как отсутствие явления.

## Связь с презентационной рамкой

Большая презентация Института «Царьград» включает профилактику абортов и разводов в число основных мер, способствующих достижению целевых показателей рождаемости. Поэтому модуль «Аборты» выполняет функцию мониторинга базового индикаторного поля для обсуждения профилактических мер.
