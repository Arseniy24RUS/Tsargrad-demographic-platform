# Third Party Notices

Платформа хранит runtime-библиотеки локально в `docs/assets/vendor/`.

## Plotly.js

- Файл: `docs/assets/vendor/plotly/plotly.min.js`
- Версия: 3.3.1
- Лицензия: MIT
- Назначение: графики на страницах рождаемости, расселения, инфраструктуры, собственного дома, маткапитала, ипотеки, выплат, браков и абортов.

## Three.js

- Файлы: `docs/assets/vendor/three/three.module.js`, `docs/assets/vendor/three/OrbitControls.js`
- Версия: r160
- Лицензия: MIT
- Назначение: процедурная трёхмерная сцена усадьбы, проектные размерные линии на земле, линия высоты дома и управление камерой. Внешние 3D-модели и удалённые текстуры не используются.

## Node / Playwright

`@playwright/test` используется только для разработки и QA. Команды `npm run test:smoke` и `npm run test:responsive` открывают локальный сервер, проверяют визуальные состояния, интерактивность, responsive-матрицу и contact sheets. В runtime платформы Node и Playwright не загружаются.

## OpenStreetMap

- Назначение: офлайн-подготовка инфраструктурных расстояний для страницы `Инфраструктура`.
- Runtime: браузер не обращается к серверам OpenStreetMap, Overpass или внешним геосервисам; опубликованы только компактные локальные JSON.
- Лицензия данных: Open Database License. Требуется сохранять атрибуцию OpenStreetMap contributors при дальнейшем распространении производных данных.

## Маткапитал

Страница `Маткапитал` использует уже локально поставляемый Plotly из `docs/assets/vendor/plotly/plotly.min.js`; новых внешних runtime-библиотек и сетевых запросов модуль не добавляет.

## Браки и аборты

Страницы `Браки` и `Аборты` используют уже локально поставляемый Plotly из `docs/assets/vendor/plotly/plotly.min.js` для трендовых графиков. Картограммы субъектов строятся собственным SVG-кодом из локальных GeoJSON; новых внешних runtime-библиотек, CDN или удалённых шрифтов страницы не добавляют.
