# Политика внешних библиотек

Runtime-платформа должна быть самодостаточной. Все JS-библиотеки, нужные браузеру при открытии страницы, должны храниться локально в `docs/assets/vendor/`.

## Разрешённые локальные библиотеки

- Plotly.js — графики.
- Three.js — трёхмерная усадьба.
- OrbitControls — управление камерой Three.js.
- SheetJS CE — только если сохранён XLSX-экспорт; предпочтительно заменить на скачивание таблицы в CSV.

## Документирование

Для каждой библиотеки в `docs/THIRD_PARTY_NOTICES.md` указать:

- название;
- версия;
- сайт/репозиторий;
- лицензия;
- локальные файлы;
- дата добавления.

## Запрещено

- Подключать CDN в HTML.
- Использовать внешние шрифты.
- Загружать текстуры с внешних доменов.
- Делать runtime-запросы к GitHub Raw.
