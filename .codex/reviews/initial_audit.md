# Initial Audit

Исходное состояние не было релизным:

- runtime использовал GitHub Raw / CDN-зависимости;
- авторский прогноз не был сохранён как локальный слой;
- Three.js импортировался не полностью локально;
- навигация не имела нумерации `01-05`;
- СКР не разделял режимы Россия / федеральный округ / субъект РФ;
- отсутствовали обязательные документы `METHODOLOGY.md`, `DATA_MANIFEST.md`, `THIRD_PARTY_NOTICES.md`;
- Playwright smoke не мог пройти как полноценная автономная проверка.

Решение: выполнить единый релизный цикл с локализацией данных, UI-правками, чекерами и Browser QA.
