# 07. Страница Усадьба

## Цель

Довести 3D-визуализацию до презентационного уровня.

## Минимум визуальных объектов

- участок;
- дорожки;
- ограждение;
- деревья;
- сад;
- стартовый дом;
- двускатная крыша;
- окна;
- дверь;
- крыльцо;
- детские модули;
- модуль старшего поколения;
- рабочий модуль;
- модуль общих пространств.

## Технические требования

- Three.js локальный.
- OrbitControls локальный.
- Fallback для отсутствия WebGL.
- Плавное появление новых модулей.
- Визуальная площадь соответствует расчётной.
