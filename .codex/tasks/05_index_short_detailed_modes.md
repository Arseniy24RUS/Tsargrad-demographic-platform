# 05. Краткий и подробный вид первой страницы

## Цель

Разгрузить страницу СКР для руководительского показа.

## Краткий вид по умолчанию

Показывать:

1. карту / картограмму;
2. режим анализа;
3. главный график СКР;
4. 4–6 KPI;
5. краткий блок `желаемое — ожидаемое — фактическое`;
6. управленческий вывод одной фразой.

## Подробный вид

Показывать:

1. РПН-2022;
2. РПН-2012/2017;
3. жилищные барьеры;
4. таблицы;
5. методику.

Региональную ретроспективу РПН свернуть по умолчанию.
