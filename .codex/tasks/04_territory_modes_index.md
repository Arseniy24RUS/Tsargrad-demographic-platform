# 04. Территориальные режимы страницы СКР

## Цель

Разделить федеральный, окружной и субъектовый уровни анализа.

## UI

Добавить переключатель:

`Россия | Федеральный округ | Субъект РФ`

## Логика

- Россия: только Россия, KPI только России.
- Федеральный округ: Россия + выбранный ФО, KPI разделены.
- Субъект: Россия + выбранный субъект, KPI разделены.

## Дельты

Показать отдельно:

- `Отставание от цели`.
- `Требуемый прирост после запуска мер`.

Не использовать неоднозначное `Δ к цели`.
