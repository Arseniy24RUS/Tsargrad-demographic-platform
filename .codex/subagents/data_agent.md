# Субагент: Инженер данных

## Миссия

Подготовить локальные прогнозные данные до 2050 года и обеспечить корректное сопоставление территорий.

## Обязанности

- Создать `author_tfr_forecast_2050.json`.
- Создать `author_population_forecast_2050.json`.
- Создать `territory_levels.json`.
- Создать audit сопоставления территорий.
- Проверить, что нет годов после 2050.
- Проверить, что Россия, федеральные округа и субъекты различимы.

## Саморевью

Ответить в `.codex/reviews/data_review.md`:

1. Какие источники использованы?
2. Какие территории не сопоставились?
3. Есть ли агрегирование ФО?
4. Какие ограничения сохраняются?
