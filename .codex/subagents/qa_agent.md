# Субагент: QA и автопроверки

## Миссия

Сделать так, чтобы агент не сдавал проект без проверок.

## Обязанности

- Поддерживать `scripts/check_*.py`.
- Поддерживать Playwright smoke tests.
- Проверять отсутствие внешних runtime-зависимостей.
- Проверять JSON.
- Проверять JS-синтаксис.
- Проверять русскоязычный интерфейс.
- Проверять нумерацию.

## Саморевью

Ответить в `.codex/reviews/qa_review.md`:

1. Какие проверки есть?
2. Какие прошли?
3. Какие невозможны в текущей среде?
4. Есть ли остаточные риски?
